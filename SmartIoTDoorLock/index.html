Smart IoT Door Lock

This image was generated using AI

This project demonstrates a Smart IoT Door Lock system built with Arduino and an RFID module, controlled and monitored using Python via serial communication.

Key Features:
  • RFID-based secure door access
  • Python script to read RFID data from Arduino
  • Serial communication between devices
  • Can be extended for logging and alerts

Hardware Used: Arduino UNO, RFID Reader (RC522), Servo Motor, Breadboard, Jump Wires.

Software: Python (pyserial), Arduino IDE

Status: Prototype tested and working locally. Cloud integration and mobile app support in progress.
